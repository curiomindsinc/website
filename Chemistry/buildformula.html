How to Play
BUILD FORMULA
1
A compound name appears. Select the correct ions to build it.
2
Tap multiple tiles of the same ion to balance charges (e.g. 1x Mg2+ + 2x Cl-).
3
When charges balance (neutral), tap Check to confirm.
SCORING & TIMING
!
35s per question. 4 min total game. Answer faster and select more ions for bigger rewards!

⚡ Speed Bonus — answer faster for more points!

Answer in 0–5s → Speed x4 🔥🔥

Answer in 5–10s → Speed x3 🔥

Answer in 10–17s → Speed x2.5

Answer in 17–24s → Speed x2

Answer in 24–30s → Speed x1.5

Answer after 30s → Speed x1

🧪 Complexity Bonus — harder formulas = more points!

2 ions total (e.g. NaCl) → Complexity x1

3 ions total (e.g. MgCl₂) → Complexity x2

4–5 ions total (e.g. Al₂O₃) → Complexity x3

6+ ions total → Complexity x4 💎

🔥 Streak Bonus — keep getting them right!

Each correct answer in a row adds +3 base points. Hit 5 in a row for FIRE mode!
